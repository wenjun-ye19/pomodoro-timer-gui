# Stoic Focus Timer GUI

A clean, cross-desktop timer built with Python and Tkinter. Designed for focus and simplicity.

## ✨ Features

- **Visual Modes**: Distinct colors for Work (Cyan) and Break (Yellow).
- **Controls**: Start, Pause, Reset functionality.
- **Alerts**: Sound and Popup notification when time is up.
- **Cross-Platform**: Runs on Windows, macOS, and Linux.

## 🚀 How to Run

```bash
python main.py
```
